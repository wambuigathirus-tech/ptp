# Priest the Poet - Samuel Njoroge Portfolio

A modern, responsive portfolio website showcasing Samuel Njoroge, professionally known as "Priest the Poet" - a multifaceted creative professional and community builder in Kenya's Orange Economy.

## 🌟 About the Artist

Samuel Njoroge is a leader in Kenya's creative space, combining art with business and community leadership. Known as "Priest the Poet," he delivers socially conscious spoken word performances while mentoring emerging artists and fostering creative communities through his work at Baraza Media Lab and various creative initiatives.

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No build tools or dependencies required

### Installation & Usage

1. **Clone or Download:**
   ```bash
   git clone https://github.com/yourusername/Priest-The-Poet-Portfolio.git
   cd Priest-The-Poet-Portfolio
   ```

2. **Open in Browser:**
   - Double-click `index.html` or
   - Use a local server: `python -m http.server 8000`
   - Navigate to `http://localhost:8000`

3. **Explore Features:**
   - Scroll through sections with smooth animations
   - Click navigation links for instant scrolling
   - View gallery with hover effects
   - Submit contact form (demo functionality)
